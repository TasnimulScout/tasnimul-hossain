Flower Shop Management System
Java JavaFX (GUI) OOP File Handling

A comprehensive desktop application designed to manage a flower boutique. It features separate dashboards for Admin and Customers with secure login and data persistence.

  • Admin Role: Manage inventory (Add/Update/Delete flowers) and view/cancel orders.
  • Customer Role: Browse available flowers, place orders, and track order history.
  • Data Persistence: All information is saved and loaded from text files using Custom File Handlers.
  • OOP Principles: Implemented Inheritance (User -> Admin/Customer) and Encapsulation.
